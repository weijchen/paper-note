Neural Machine Translation by Jointly Learning to Align and Translate

作者: Dzmitry Bahdanau, Kyunghyun Cho, Yoshua Bengio

研究目的

  • 為了解決 auto-encoder 在處理 fixed-length vector 時的問題，特別在長句子的 case

研究方式

  • 提出稱為 (soft-)alignments 的方式 -> learns to align and translate jointly

    • 每次翻譯的時候 (whenever generating a word)，(soft-)search 句子中最相關的位置
    • Encodes the input sentence into a sequence of vectors and chooses a subset of these vectors

  • 文中針對英文對法文的翻譯問題進行測試

  • Common Encoder-Decoder (via RNN) ref-1, ref-2:

    • Encoder:
      • 透過當下的 input 𝑥𝑡xt 以及前一個時間點的 hidden state ℎ𝑡−1ht−1 來定義當下的 hidden state,
        ℎ𝑡=𝑓(𝑥𝑡,ℎ𝑡−1)ht=f(xt,ht−1)
      • 再轉換所有的 hidden state 成 context vector 𝑐c,
        𝑐=𝑞(ℎ1,...,ℎ𝑇𝑥)c=q(h1,...,hTx)
      • 𝑓f, 𝑞q 為 nonlinear function
    • Decoder:
      • 利用 Encoder 中所得的 context vector 𝑐c 以及過往所 predict 的字 𝑦=𝑦1,...,𝑦𝑇𝑦y=y1,...,yTy 進行轉換，求出機率最高的結果 𝑝(𝑦)p(y),
        𝑝(𝑦)=∏𝑡=1𝑇𝑝(𝑦𝑡|{𝑦1,...,𝑦𝑡−1},𝑐)p(y)=∏t=1Tp(yt|{y1,...,yt−1},c)
         其中
        𝑝(𝑦𝑡|{𝑦1,...,𝑦𝑡−1},𝑐)=𝑔(𝑦𝑡−1,𝑠𝑡,𝑐)with an RNNp(yt|{y1,...,yt−1},c)=g(yt−1,st,c)with an RNN
      • 𝑔g is a nonlinear function, 𝑠𝑡st is the hidden state of the RNN.

  • Proposed approach:

    • Encoder - Bidirectional RNN ref-3
      ℎ𝑗=[ℎ⊺𝑗→;ℎ⊺𝑗←]hj=[hj⊺→;hj⊺←]
    • Decoder
      • Conditional probability is
        𝑝(𝑦𝑡|{𝑦1,...,𝑦𝑡−1},𝑐𝑖)=𝑔(𝑦𝑡−1,𝑠𝑡,𝑐)p(yt|{y1,...,yt−1},ci)=g(yt−1,st,c)
         and 𝑠𝑖si is an RNN hidden state for time 𝑖i, computed by
        𝑠𝑖=𝑓(𝑠𝑖−1,𝑦𝑖−1,𝑐𝑖)si=f(si−1,yi−1,ci)
         跟前面不同的是， 𝑦𝑖yi 會對應到一個獨立的 context vector 𝑐𝑖ci
      • each ℎ𝑖hi contains information about the whole input sequence with a strong focus on the parts surrouding the 𝑖i-th word of the input sequence.
        𝑐𝑖=∑𝑗=1𝑇𝑥𝑎𝑖𝑗ℎ𝑗ci=∑j=1Txaijhj
         𝑎𝑖𝑗aij is the weight of each tation ℎ𝑗hj,
        𝑎𝑖𝑗=exp(𝑒𝑖𝑗)∑𝑇𝑥𝑘=1exp(𝑒𝑖𝑘)where𝑒𝑖𝑗=𝑎(𝑠𝑖−1,ℎ𝑗)aij=exp⁡(eij)∑k=1Txexp⁡(eik)whereeij=a(si−1,hj)
      • 𝑒𝑖𝑗eij 的分數根據 𝑠𝑖−1si−1 (RNN hidden state) 以及 ℎ𝑗hj (𝑗j-th hidden layer) 而定 -> 定義了 position 𝑗j 附近的 input 跟 position 𝑖i 的 output 間的關係
      • 直觀上，這裡可理解成計算期望值的概念 (expected annotation)，𝑎𝑖𝑗aij 是 𝑦𝑖yi 可轉換成 𝑥𝑗xj 的機率，而 𝑐𝑖ci 就會是在這個機率下的 expected annotation

    • 用了兩種模型: RNN Encoder-Decoder, Proposed approach
